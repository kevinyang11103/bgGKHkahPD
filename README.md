# 项目简介

## 核心功能
- 用户登录与注册
- 商品分类展示
- 联系卖家
- 发布购买需求
- 发布出售信息
- 求购专区浏览
- 物品评论功能
- 后台管理首页
- 菜单管理
- 角色管理
- 用户管理
- 日志管理
- 数据备份
- 商品分类管理
- 商品信息管理
- 求购信息管理
- 学生信息管理
- 评论信息管理
- 举报信息管理
- 新闻公告管理
- 友情链接管理
- 站点设置管理

## 适用场景
- 校园内部学生之间进行二手物品交易
- 简化交易流程，提高物品利用效率
- 提供一个便捷的平台用于发布和查找二手物品
- 增强校园社区互动和资源共享

## 技术特点
- 基于 Spring Boot 开发，保证了系统的稳定性和高效性
- 前后端分离，易于维护和扩展
- 采用模块化设计，功能清晰，易于管理
